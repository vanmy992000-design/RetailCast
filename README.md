## ⚙️ Setup (first time)

### 1. Clone / download project folder

### 2. Create virtual environment (recommended)
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Mac/Linux
source venv/bin/activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

---

## 🚀 Run the project

### Option A — Run full pipeline (all 3 steps at once)
```bash
python run_pipeline.py
```

### Option B — Run each step individually
```bash
python run_pipeline.py --step 1    # Data cleaning + feature engineering
python run_pipeline.py --step 2    # EDA + 19 charts
python run_pipeline.py --step 3    # 3 forecasting models
```

### Option C — Run each step script directly
```bash
python src/step1_data_pipeline.py
python src/step2_eda.py
python src/step3_forecasting.py
```

## 🤖 Models

| Model | Library | Best for |
|-------|---------|---------|
| SARIMA | `statsmodels` | Seasonal patterns (monthly cycles) |
| Prophet | `prophet` | Trend detection + changepoints |

## ⚠️ Lỗi thường gặp

### `ModuleNotFoundError: No module named 'prophet'`

Prophet cần cài riêng. Thử theo thứ tự:

**Cách 1 — pip thông thường:**
```bash
pip install prophet
```

**Cách 2 — Nếu cách 1 lỗi trên Windows:**
```bash
pip install pystan==2.19.1.1
pip install prophet
```

**Cách 3 — Dùng conda (khuyên dùng cho Windows):**
```bash
conda install -c conda-forge prophet
```

**Cách 4 — Bỏ qua Prophet, chỉ chạy SARIMA + XGBoost:**
```bash
python run_pipeline.py --skip-prophet
```

### `ModuleNotFoundError: No module named 'xgboost'`
```bash
pip install xgboost
```

### `ModuleNotFoundError: No module named 'statsmodels'`
```bash
pip install statsmodels
```
